#  Explore Nepal Website

##  Project Overview
Explore Nepal is a multi-page interactive website developed using HTML, CSS, and JavaScript.
It allows users to explore Nepal’s culture, mountains, food, and popular places through an
engaging and user-friendly interface.

---

##  Features
- Multi-page navigation system
- Interactive explore section (mountains, places, food, culture)
- Image-based content display
- Form validation for contact page
- Creator section with reveal/hide functionality
- Responsive and simple UI design

---

##  Technologies Used
- HTML5
- CSS3
- JavaScript
- jQuery
